🎯 Neden Firebase?
Firebase Google'ın ücretsiz bulut veritabanı. Sunuc Firebase Google'ın ücretsiz bulut veritabanı. Sunucu kurmanıza gerek yok. Linki paylaştığınızda tüm kullanıcılar aynı veriye gerçek zamanlı erişir. 5–10 kullanıcı ve aylık veri girişi için ücretsiz plan tamamen yeterli.
Kullanıcı Listesi Şablonu
const KULLANICILAR = {
  "yonetici@sirket.com":  { ad: "Genel Müdür",    rol: "admin", depo: null  },
  "d01@sirket.com":       { ad: "D-01 Sorumlusu", rol: "depo",  depo: "D-01" },
  // rol:"admin" → tüm depolar, rapor ve sıralama
  // rol:"depo"  → yalnızca kendi deposu
};
